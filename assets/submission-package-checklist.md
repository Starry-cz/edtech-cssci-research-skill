# 教育技术学论文投稿包核对表

## 官方规则

- [ ] 已记录目标期刊唯一官网、作者指南版本与核验日期。
- [ ] CSSCI/核心目录、栏目、模板和投稿入口均来自当前官方来源。
- [ ] 未把近刊惯例或第三方网页写成期刊明文要求。

## 内容与证据

- [ ] 问题—理论—模型—证据—实践—结论可互相追溯。
- [ ] 数据、分析、图表、摘要和结论使用同一最终版本。
- [ ] 相关、预测、干预和因果措辞符合研究设计。
- [ ] 伦理、权限、隐私、公平、利益冲突与作者贡献真实完整。

## 引文与参考文献

- [ ] 正文引文与文后条目双向一致，无重复或错指。
- [ ] 关键文献已核对原始页面，直接引语含准确页码。
- [ ] 引文体例、标点和条目字段符合目标期刊最新规则。
- [ ] `[待核验]`、占位符、批注和内部标签已清零或获得作者确认。

## 文件与匿名化

- [ ] 主稿、匿名稿、作者信息页、图表、附录和声明齐全。
- [ ] 匿名稿已清除姓名、单位、基金、致谢、修订记录、属性和可识别链接。
- [ ] DOCX/PDF 已实际渲染检查，无裁切、乱码、错页、孤行和字体替换。
- [ ] 图表数字、单位、图注、来源与正文一致，灰度和缩放后仍可读。

## 投稿材料

- [ ] 中英文题目、摘要和关键词信息一致。
- [ ] 投稿信、回应函及补充材料的版本和定位准确。
- [ ] 数据/代码可用性、AI 工具披露及其他声明按官方要求填写。
- [ ] 作者本人将核对账户、身份信息、作者顺序、协议签署和最终提交。
