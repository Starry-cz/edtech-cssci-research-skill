# 教育研究统计报告与图表证据门

用于起草或审查方法、结果、图注、统计分析小节和统计类审稿意见。它只帮助报告和发现风险；没有数据、设计或分析计划时，不替代统计重分析或虚构检验结果。

## 一、三项先行判断

在选择检验或改写结果前，明确：

1. **测量了什么**：构念、量尺、评分/编码/日志生成方式与测量时点；
2. **分析了谁**：独立分析单位是学习者、班级、教师、学校、课程还是时间窗口，技术重复和重复观测如何处理；
3. **主张什么**：描述、关联、差异、预测、变化、干预效果或机制分别需要不同设计。

信息不足时列 `AUTHOR_INPUT_NEEDED` 或“待补材料”，不能猜测样本量、模型、软件、随机化、排除规则、p 值或置信区间。

## 二、优先检查的高风险问题

| 风险 | 文字或设计信号 | 修复方向 |
| --- | --- | --- |
| 伪重复/层级错配 | 把同一学生多次作答、班内学生、视频片段或技术编码当作相互独立的 `n`。 | 聚合到独立单位，或采用相称的多层/混合/重复测量模型。 |
| 多重比较不透明 | 多量表、时间点、组别、模型或事后比较中只报告显著项。 | 界定比较家族，区分主要/探索性分析，采用合适校正或报告其缺失。 |
| 交互推断错误 | “A 显著、B 不显著，所以二者不同”。 | 直接检验交互或比较效应差异与区间。 |
| 仅显著性叙述 | 只报星号、阈值或“显著提升”。 | 报告估计、效应量、区间、样本及教育实际意义。 |
| 小样本/不稳定强主张 | 样本很小或区间很宽，却下机制或推广结论。 | 说明精度限制，降低结论并优先报告分布。 |
| 异常值和缺失不透明 | 点被删除、样本量变化或缺失处理无规则。 | 报告规则、数量、时点与包含/替代处理下的稳健性。 |
| 相关、回归或路径越界 | 关联写成影响、机制、预测或因果。 | 回到设计和识别条件，使用关联表述或补足识别。 |

预测建模另读取 `interpretable-machine-learning.md`；多层、纵向、SEM、准实验等具体设计先读取 `education-data-analysis.md`。本文件不把自然科学实验中的“重复次数”概念直接套到教育场景。

## 三、结果与图注的最小报告字段

| 位置 | 必须让读者知道 | 不能省略为 |
| --- | --- | --- |
| 方法 | 样本形成、独立单位、组别/层级/时间、排除与缺失、分析模型和软件版本（若已知）。 | “采用 SPSS/AMOS/R 分析”。 |
| 结果 | 比较/参数、估计或效应量、区间/不确定性、样本量和结论边界。 | “结果显著，证明有效”。 |
| 图注 | 面板含义、`n` 的定义、误差线/箱线含义、统计比较、校正状态、单位和时间范围。 | 仅放 `*p<.05` 或“差异显著”。 |
| 表注 | 指标方向、量纲、有效位数、缺失/加权说明与模型口径。 | 混合不同统计口径却不说明。 |

一张表图只传达一个主要判断。优先展示原始分布、点区间、轨迹或条件效应；柱形图与星号不能替代分布和不确定性。若同一条件在多面板比较，坐标、单位、配色和样本定义须可比。

## 四、统计报告输出

默认分为四块：

1. **审查范围与材料边界**；
2. **设计与独立单位读出**；
3. **P0/P1/P2 风险**：每项给材料证据、为何重要与修复动作；
4. **可粘贴修订文本 + AUTHOR_INPUT_NEEDED**。

P0 指可能改变主要结论的独立性、比较家族、设计错配或结果不可复核问题；P1 指重要的报告或解释风险；P2 指图注、标记、单位、精度和可读性问题。只有用户提供足够事实时才输出可粘贴的统计文本；否则先输出需要补充的事实问题。
