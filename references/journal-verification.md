# CSSCI 期刊与投稿规范核验流程

仅在用户询问期刊匹配、CSSCI资格、投稿格式、栏目或投稿策略时阅读本文件。期刊目录、栏目、格式和系统要求会变化，不能依赖静态记忆。

## 核验顺序

1. 访问目标期刊官网或编辑部的最新公告，确认栏目范围、投稿系统、作者指南、匿名要求、篇幅与参考文献规则。
2. 对 CSSCI 或北大核心收录情况，查询权威发布机构或用户指定数据库的当前版本，并标注查询日期。
3. 比较论文的问题、对象、方法与该刊近期已刊文章的主题和证据偏好；这只能用于匹配判断，不能替代官方要求。
4. 无法核验时，明确说明“未核验”，请用户提供作者指南或官网链接；不得虚构录用率、审稿周期、版面费或格式细节。

## 投稿前最小清单

- 题目、摘要、关键词、作者信息、匿名稿与基金信息满足目标期刊最新要求。
- 参考文献逐条可定位，文内引注与文后条目一致。
- 研究对象、样本、数据权限、伦理与知情同意说明与实际研究相符。
- 研究结论、贡献和建议未超出方法、样本及情境边界。
