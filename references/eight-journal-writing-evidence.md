# 八刊写作证据台账与诊断规则

本文件把《开放教育研究》《远程教育杂志》《中国电化教育》《现代远程教育研究》《电化教育研究》《现代教育技术》《现代远距离教育》《中国远程教育》作为**写作与论证的样本库**。它不是投稿资格清单，也不表示八刊在任何时点具有相同的 CSSCI 身份、栏目或格式；投稿前仍须以目标期刊当期官方网站核验。

样本只用于识别可迁移的论证组织：问题如何进入、理论怎样进入方法、结果与讨论如何分工、图表承担什么判断。不得仿写作者句式、复制摘要结构，或以样本中的强因果措辞为当前研究的证据许可。

## 使用规则

1. 先按研究类型选取同类样本：预测研究对照预测/学习分析样本，设计研究对照设计/实践样本，不能用理论文章的标题强度覆盖横断面或预测设计。
2. 摘要只学习信息的**功能顺序**：具体问题或缺口—对象和设计—最小发现—相称含义；最终仍以研究自身的已核验材料决定内容。
3. 图表先说明回答哪一个研究问题、比较什么、误差或不确定性如何呈现；不能从版式、配色或图数量推出证据强度。
4. 题名、期刊年份、链接和获取状态只作检索锚点。使用具体样本前重新打开来源页核对题录、版本和全文可得性。

## 可迁移的共性规则

| 环节 | 可迁移规则 | 不可迁移的误读 |
| --- | --- | --- |
| 问题入口 | 从学习、教学、评价、教师发展或治理中的具体矛盾进入，再说明现有解释或设计何处不足。 | 用政策背景、技术热点或系统名称直接充当研究问题。 |
| 摘要 | 连续段落中兑现研究对象、设计功能、关键发现和有限含义；方法信息服务问题，不按软件运行顺序堆叠。 | 复制某文的句长、条目数、动词或强结论。 |
| 理论—方法 | 理论需落到构念、特征/编码、比较条件、设计原则或解释竞争项。 | 综述中出现理论名称，就称研究已具理论框架。 |
| 结果—讨论 | 结果报告观察到的比较、模式和不确定性；讨论才处理与文献的关系、替代解释和实践含义。 | 让一张图、一次显著性检验或一个模型指标同时承担机制、效果和推广结论。 |
| 图表 | 每张表图只承担一个主要判断，并在图题、图注和正文中交代对象、数据、模型/比较与限制。 | 用框架箭头、模型排名或漂亮曲线替代过程、识别或复现证据。 |
| 贡献 | 用“澄清了什么、在何种条件下、凭什么证据”描述最小贡献。 | 把系统上线、预测准确率、政策价值或研究样本数量直接写成理论创新。 |

## 专项对照：可解释教育人工智能与预测研究

以下四篇用户提供论文只用于内部规则提炼，不上传全文、不复制段落，也不把其中的研究结论迁移到其他数据。

| 对照材料 | 应保留的规则 | 不能据此放行的表述 |
| --- | --- | --- |
| 罗杨洋、韩锡斌（2022）混合课程成绩预测解释性研究 | 解释模型需交代预测关系、适用条件与决策可信度，而非只有特征排名。 | 特征归因等同于对学习结果的净效应。 |
| 钱阿剑、赵茜（2026）创造性思维研究 | 可在摘要中把数据、模型解释、异质性与发现按论证功能压缩。 | 把机器学习发现直接命名为形成机制或跨文化因果。 |

## 八刊风格诊断输出

当用户提出“按八刊风格诊断”时，输出四项：

1. 当前稿件类型与应对照的样本类型；
2. 问题、理论—方法、结果—讨论、图表四个接口的匹配/缺口；
3. 可迁移的组织规则与不可迁移的措辞或结论；
4. 当前发布状态（`阻断`、`探索性可写` 或 `投稿就绪`）及修复路径。

不得输出“某刊一定偏好”“模仿某作者即可录用”或未经当前官方核验的格式与收录结论。
