# 数据采集、测量与分析审查

在审查抽样、量表、日志、编码、数据清理、统计检验或模型解释时读取。涉及教育数据分析的完整执行流程、结果报告、可视化和复现时，同时读取 `education-data-analysis.md`。具体结论服从研究问题、数据结构和目标期刊要求，不机械套用固定样本量或统计阈值。

## 分析单位与抽样

- 区分总体、抽样框、样本、抽样过程和分析单位；教育技术研究的单位可能是学习者、教师、班级、课程、学校、行为事件、帖子、任务或时间窗口。
- 识别嵌套与重复测量结构。学生嵌套班级/学校、行为嵌套个人时，普通独立观测模型可能低估不确定性。
- 概率抽样的总体推断、便利/目的样本的机制探索、案例研究的分析性推广应使用不同表述。
- 样本量按研究设计、功效或精度、预期效应、变量/参数数量、聚类、流失、可获得性和伦理负担论证；不使用无条件的统一数字。
- 报告招募、纳排、流失、缺失、平台用户选择偏差和样本与目标群体差异。

## 概念化、操作化与测量

- 先定义构念及适用情境，再选择量表、行为指标、编码类别或学习产出。
- 成熟量表仍需核对语言、教育层级、技术场景、内容效度和测量不变性；改编须说明过程。
- 区分信度与效度；内部一致性不能单独证明构念效度。
- 说明定类、定序、连续、计数、时间到事件等变量类型，并让分析方法匹配。
- 平台点击、停留时间和完成率是行为代理指标，不自动等于投入、深度学习或认知参与。

## 数据生成与清理

- 保存原始数据与处理后数据的边界，记录版本、变量字典、代码本和处理规则。
- 报告重复、无效响应、异常作答、机器人流量、日志时区、事件丢失、跨设备身份合并和传感器/平台变更。
- 说明缺失机制、缺失比例、处理方法和敏感性；不默认整案删除或单一插补。
- 预先说明排除、异常值、重编码、合成分数和特征工程，避免根据结果事后选择。
- 对人工/大模型编码说明编码本、训练、双人或抽样复核、一致性和分歧处理。

## 描述与统计检验

- 先展示样本、变量分布、缺失和关键组别，再进入推断。
- 根据变量类型、独立/配对结构、组数、分布、方差、聚类与研究问题选择检验；必要时采用稳健或非参数方法。
- 报告效应量、置信区间/不确定性和实际教育意义，不只报告显著性。
- 多重比较、探索性分析和多模型尝试应透明报告并控制误报风险。

## 回归、结构方程与预测

- 模型选择和预测变量纳入须有理论或明确预测目标，不以“数据里有变量”为理由。
- 检查线性、共线性、残差、分离、拟合、过拟合、测量模型和其他与模型相关的假设/诊断。
- 区分解释模型与预测模型；预测任务报告训练/验证隔离、基线、泛化、校准和公平性。
- 中介、调节和结构路径需要时间顺序、理论与识别条件；横断面 SEM 不自动建立因果机制。
- 可解释机器学习任务同时读取 `interpretable-machine-learning.md`；数据清理、插补、编码、标准化、降维和特征选择必须在训练折内拟合。

## 教育干预与因果边界

- 报告分配方式、对照条件、前测、干预成分、实施忠实度、污染、流失和分析策略。
- 准实验说明可比性、识别假设、协变量/匹配/断点/差分策略和稳健性。
- 相关、回归、日志模式和机器学习预测不得直接写成“导致、促进、提升”。

## 诊断输出

按需标记 `Sampling Gap`、`Operationalization Gap`、`Measurement Reliability Gap`、`Measurement Validity Gap`、`Data Provenance Gap`、`Data Cleaning Gap`、`Statistical Test Mismatch`、`Regression Strategy Gap` 或 `Causal Overclaim`，并提供当前安全表述和需要补充的证据。
